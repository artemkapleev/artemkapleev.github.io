# 📚 Домашние работы — GitHub Pages

Статичный сайт для хранения и просмотра домашних работ в формате PDF.

## Структура репозитория

```
/
├── index.html          ← главная страница (редактируется для добавления работ)
├── README.md
└── homeworks/          ← папка с PDF-файлами
    ├── 2025-04-10-drobi.pdf
    ├── 2025-04-07-sochinenie.pdf
    └── ...
```

## Как добавить новую работу

### 1. Добавьте PDF в папку `homeworks/`

Рекомендуемый формат имени файла:
```
YYYY-MM-DD-название.pdf
```

Пример: `2025-04-15-tablitsa-umnozheniya.pdf`

### 2. Откройте `index.html` и найдите массив `homeworks`

```javascript
const homeworks = [
  // ← добавьте новую запись сюда, в начало массива
  {
    date: "2025-04-15",
    name: "Таблица умножения",
    subject: "Математика",
    file: "homeworks/2025-04-15-tablitsa-umnozheniya.pdf"
  },
  // ... остальные работы
];
```

### 3. Сделайте git push

```bash
git add .
git commit -m "Добавлена домашняя работа: Таблица умножения"
git push
```

Сайт обновится автоматически через 1–2 минуты.

## Настройка GitHub Pages

1. Перейдите в **Settings → Pages** репозитория
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Сохраните — сайт будет доступен по адресу `https://username.github.io/repo-name/`

## Предметы (subject)

Можно использовать любые названия, например:
- `Математика`
- `Русский язык`
- `Окружающий мир`
- `Английский язык`
- `Литература`
- `Рисование`

## Функции сайта

- 📋 Список работ, отсортированных по дате (новые первые)
- 👁 Открытие PDF в новой вкладке
- ⬇️ Скачивание файла
- 🖨 Кнопка печати
- 🌙 / ☀️ Переключение тёмной и светлой темы
- 📱 Адаптивный дизайн для телефона и компьютера
- 📊 Статистика: всего работ, в этом месяце, дата последней
